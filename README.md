Cogniai Landing Page

A modern, responsive landing page built with Next.js 14, TypeScript, and Tailwind CSS.

🚀 Features

⚡ Optimized with Next.js App Router

🎨 Styled using Tailwind CSS

📱 Fully responsive design

🧩 Component-based architecture (app/ and components/ folders)

🌙 Easy to extend and customize

📂 Project Structure
cogniai-landing/
│── app/               # Next.js app directory (pages, layout, globals.css)
│── components/        # Reusable UI components
│   ├── sections/      # Landing page sections (About, Pricing, etc.)
│   └── ui/            # Shared UI elements (Button, Card, etc.)
│── lib/               # Utility functions
│── public/            # Static assets (if any)
│── styles/            # Global styles (Tailwind, CSS)
│── package.json       # Dependencies and scripts
│── tsconfig.json      # TypeScript configuration
│── tailwind.config.ts # Tailwind configuration

🛠️ Tech Stack

Framework: Next.js 14

Language: TypeScript

Styling: Tailwind CSS + tailwindcss-animate

Deployment: Vercel

⚡ Getting Started
1. Clone the repository
git clone https://github.com/Srijadaripelli/cogniai-landing.git
cd cogniai-landing

2. Install dependencies
npm install

3. Run development server
npm run dev


App will be available at: 👉 http://localhost:3000

4. Build for production
npm run build
npm run start
